# 超自然自动备份

自动备份目标游戏数据，防止游戏突然自动更新导致辅助工具不适配。

---

## ⚠️ 使用前必读

### 🔑 强烈建议：定期清理游戏缓存

**这是决定备份大小的关键操作！**

> 游戏主界面 → 设置 → 画面设置 → 清理缓存

| 状态 | files 文件夹大小 | 备份占用空间 |
|------|-----------------|-------------|
| 未清理缓存 | 5~8G | 备份也占用 5~8G |
| 清理缓存后 | **小于 1G** | 备份仅占用几百MB |

清理缓存后再备份，空间占用减少 **80% 以上**，备份和恢复速度也快很多。

---

### 关于存储空间

| 备份类型 | 说明 | 保留数量 |
|----------|------|----------|
| 快照（未压缩） | 源文件大小 = 当前 files 大小 | 最多 2 个 |
| 压缩备份 | 约为源文件的 50%~70% | 最多 2 个 |

- **清理缓存后**：全部备份总计约 **1~2G**
- **未清理缓存**：全部备份总计可能达 **20~30G**

请确保手机有足够剩余空间。

---

## 功能说明

### 自动备份
打开游戏时，模块自动创建快照；关闭游戏后，自动压缩为备份文件。

### 手动操作
在模块管理器中点击"运行"，按音量键选择：
- **音量+** → 立即创建快照
- **音量-** → 恢复最新快照（需二次确认）

---

## 安装

1. 下载模块 ZIP 包
2. 在 KernelSU / Magisk 中刷入
3. 重启手机

---

## 常见问题

**Q: 备份文件在哪里？**
A: 快照和备份均存储在游戏数据目录下的隐藏文件夹中。

**Q: 如何释放空间？**
A: 模块自动保留最近 2 个备份，超出自动删除。也可手动删除旧备份。

**Q: 恢复后游戏会回档吗？**
A: 会恢复到备份时的游戏状态，请谨慎操作。
