# 更新日志

## v2.0 - 2026-05-14

### 🎉 重大更新：全新 WebUI 控制面板
- 模块现支持在 KernelSU 管理器中直接打开 WebUI 界面
- 无需按键，点击按钮即可完成所有操作
- 界面毛玻璃风格，背景使用随机 API 高清图片
- 实时显示游戏运行状态（绿色圆点 = 运行中）

### 🚀 新增功能
- **一键断网启动游戏**：点击按钮或运行模块自动执行
  1. 自动记录当前 WiFi 和移动数据状态
  2. 关闭所有网络
  3. 启动游戏
  4. 等待 10 秒
  5. 自动恢复之前的网络状态
- **自由选择备份恢复**：弹出列表，可任选快照或压缩备份进行恢复
- **安全恢复模式**：恢复前保留旧 files，可手动删除
- **激进恢复模式**：恢复前彻底删除旧 files，干净不留残留

### 📦 备份命名优化
- 快照命名改为 `快照_20260514_143052_6.1G`，一眼看清时间和大小
- 压缩备份命名改为 `备份_20260514_143052_3.2G`
- 列表显示时间格式更友好：`2026-05-14 14:30:52`

### ✨ 优化改进
- 减少快照和备份保留数量至各 2 个，大幅节省存储空间
- 优化硬链接兼容性，快照目录移至同分区确保秒级创建
- 代码结构优化，提取公共函数减少重复逻辑
- 智能大小显示（自动转换 B/K/M/G 单位）
- 背景图片更清晰，降低遮罩透明度
- 页面支持上下滑动，修复显示不完整问题
- 守护进程开机自启，打开游戏自动快照，关闭后自动压缩

### 🛠 修复
- 修复 Android 11+ 部分设备无法访问存储路径的问题
- 修复 FUSE 文件系统硬链接失败导致备份缓慢的问题
- 修复 SELinux 上下文不匹配导致的权限错误
- 修复 WebView 内按钮点击无反应的问题
- 修复多处兼容性问题，提升稳定性

### 📄 文档
- 新增 `README.md` 详细使用说明
- 新增 `CHANGELOG.md` 更新日志
- 内置赞助方式与收款码
